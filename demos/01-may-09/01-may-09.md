# Homework — GROUP BY and HAVING

## Simple Queries

1. Display the number of countries in each continent from the `country` table in the `world` database.

2. Display the average population of countries for each continent.

3. Display the number of customers in each store from the `customer` table in the `sakila` database.

4. Display the number of films available for each movie rating, sorted by the number of films in descending order.

---

## Medium Queries

5. Display the continents whose average country population is greater than 20 million.

6. Display the total replacement cost and average replacement cost of films for each movie rating.

7. Display the districts in India (`CountryCode = 'IND'`) that contain more than 10 cities, sorted by city count in descending order.

---

## Complex Queries

8. Display, for each movie rating:

   * total number of films
   * average rental duration
   * minimum rental duration
   * maximum rental duration

   Show only those ratings whose average rental duration is at least 5 days.

9. Display the language IDs that have more than 20 films associated with them, along with the average film length for each language.

10. Display, for each continent:

* number of countries
* total population
* average life expectancy

Consider only countries whose population exceeds 1 million, and show only continents whose total population exceeds 500 million. Sort the output by total population in descending order.
